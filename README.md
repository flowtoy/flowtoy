# flowtoy

A minimal YAML-driven flow runner with dependency-aware parallel scheduling and an extension-based provider system.

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
flowtoy run config.yaml
```

For complete documentation, tutorials, and guides, visit the [flowtoy documentation](https://flowtoy.github.io/flowtoy).
